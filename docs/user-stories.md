## 📖 Histórias de Usuário

**[US01]**  
Como usuário, quero colar um link do YouTube para iniciar o processo de download.

**[US02]**  
Como usuário, quero visualizar as informações do vídeo antes de baixar, para confirmar se é o conteúdo correto.

**[US03]**  
Como usuário, quero escolher entre os formatos MP3 e MP4, para obter o tipo de mídia desejado.

**[US04]**  
Como usuário, quero selecionar a pasta de destino, para definir onde os arquivos serão salvos.

**[US05]**  
Como usuário, quero formatar automaticamente os nomes dos arquivos, para organizar melhor os downloads, incluindo:
- enumeração automática dos arquivos  
- remoção de caracteres especiais do título

**[US06]**
Como usuário, quero utilizar a aplicação diretamente no meu computador, sem depender de navegadores ou sites externos, para realizar downloads de forma mais segura e prática.

**[US07]**  
Como usuário, quero baixar múltiplos vídeos a partir de uma playlist, para evitar a necessidade de baixar cada vídeo individualmente.

**[US08]**  
Como usuário, quero visualizar o progresso do download atual, para acompanhar o andamento e estimar o tempo restante.
